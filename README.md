# AstrBot Dog Tag 插件 🐶

注册管理个人身份卡（狗牌），管理名字、生日、装备和赠送。

## 功能

- **身份注册** — 创建你的狗牌身份
- **信息管理** — 管理名字、生日、自动计算年龄
- **装备系统** — 佩戴/取下装备，自动计时，记录历史最长佩戴时间
- **装备补时** — 回溯装备已有时长，持续计时
- **装备赠送** — 群友间赠送装备，接收方可选择接受或拒绝
- **互看狗牌** — 快速查看其他群友的装备情况

## 安装

### 手动安装

将本仓库的以下文件放入 AstrBot 插件目录（`addons/plugins/`），重启 AstrBot：

- `main.py`
- `metadata.yaml`
- `_conf_schema.json`

### Git Clone

```bash
cd addons/plugins/
git clone https://github.com/KayYiny/astrbot-plugin-dogtag.git
```

## 使用指南

### 注册

首次使用需先注册：

```
/注册 张三
```

注册后即可使用其他功能。一个用户只能注册一次。

### 个人信息管理

```
/改名 李四              # 修改显示名字
/生日 2020.05.15        # 设置生日（格式：yyyy.mm.dd），自动计算年龄
/狗牌                   # 查看个人信息卡片和当前装备
/狗牌帮助                # 查看所有命令说明
```

### 装备系统

装备佩戴后自动计时，取下时显示佩戴时长并更新历史记录。可同时佩戴多个装备。

```
/装备                   # 查看当前佩戴的装备列表及已佩戴时长
/装备 手表               # 佩戴装备，开始计时
/取下 手表               # 取下装备，显示本次佩戴时长
```

取下装备时：

- 若刷新了该装备的历史最长佩戴时间 → 提示 🏆 新纪录
- 若未破纪录 → 显示当前最长记录供参考

### 装备补时

如果开始使用插件之前就已经佩戴了该装备，可以用补时命令回溯时长：

```
/装备补时 手表 5天        # 手表已佩戴 5 天，回溯后继续计时
/装备补时 徽章 1天2小时   # 支持混合单位
```

若尚未佩戴该装备，补时命令会自动装备上并设定开始时间；若已佩戴，则调整开始时间。

支持的单位：`天`、`小时`、`分`、`秒`，以及组合写法如 `1天2小时30分`。

时长自动换算规则：
```
30分 → 2小时15分 → 3天5小时 → 1周2天 → 2月15天 → 1年
```

### 赠送系统

可以将装备赠送给群友。赠送者不需要自己拥有该装备。

```
/送 @小红 手表           # 向小红赠送手表
/同意                   # 接受赠送（装备进入接收方装备栏）
/拒绝                   # 拒绝赠送
```

流程：
1. 赠送者发送 `/送 @对方 <装备名>`
2. 接收方收到通知后输入 `/同意` 或 `/拒绝`
3. 接受后装备标记来源，在信息卡片中显示：`手表（2小时）来自小红`

每人同时只能有一个待确认的赠送请求。接收方已有待确认时，新的赠送会自动被拒绝。

### 查看群友装备

一行快速查看对方当前佩戴的装备：

```
/看 @小红
🐶 小红 🎒 手表（2小时）来自张三
```

## 完整命令列表

| 命令 | 说明 |
|------|------|
| `/注册 <名字>` | 注册狗牌身份 |
| `/改名 <新名字>` | 修改显示名字 |
| `/生日 yyyy.mm.dd` | 设置生日 |
| `/装备` | 列出当前佩戴的装备 |
| `/装备 <装备名>` | 佩戴装备，开始计时 |
| `/取下 <装备名>` | 取下装备，显示时长 |
| `/装备补时 <装备名> <时长>` | 回溯已有佩戴时长 |
| `/送 @对方 <装备名>` | 赠送装备 |
| `/同意` | 接受赠送的装备 |
| `/拒绝` | 拒绝赠送的装备 |
| `/狗牌` | 查看个人信息卡片 |
| `/看 @群友` | 查看群友装备 |
| `/狗牌帮助` | 显示操作指南 |

## 数据存储

数据保存在 `data/plugin_data/astrbot-plugin-dogtag/` 目录下，每个用户一个独立的 JSON 文件。

每个文件包含以下字段：

| 字段 | 说明 |
|------|------|
| `id` | 用户唯一标识（SID） |
| `name` | 群昵称（自动同步） |
| `display_name` | 用户自定义的显示名字 |
| `birthday` | 生日 |
| `register_date` | 注册日期 |
| `equipment` | 当前佩戴的装备列表及开始时间 |
| `equipment_history` | 每种装备的历史最长佩戴记录 |
| `pending_gift` | 待确认的赠送请求 |

## 配置选项

在 AstrBot 插件面板中可调整以下配置：

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `show_register_tips` | bool | true | 未注册用户使用命令时是否提示 |
| `birthday_reminder_enabled` | bool | true | 生日当天在信息卡片显示祝福 |
| `custom_name_prefix` | string | "" | 显示名字时添加的统一前缀 |
| `max_name_length` | int | 20 | 名字最大字符数 |
| `max_equipment` | int | 0 | 同时佩戴装备数量上限（0 为无上限） |
| `max_equip_name_length` | int | 6 | 装备名字最大字符数 |
