# AstrBot Dog Tag 插件 🐶

注册管理狗牌身份，管理名字、生日和装备。

## 功能

- **身份注册** — 创建你的狗牌
- **信息管理** — 管理名字、生日、年龄
- **装备系统** — 佩戴/取下/回溯时长，自动计时
- **历史纪录** — 每种装备的最长佩戴时间
- **装备赠送** — 群友间互相赠送装备
- **互看狗牌** — 快速查看群友装备

## 安装

将本仓库所有文件放入 AstrBot 插件目录（`addons/plugins/`），重启 AstrBot。

## 命令

| 命令 | 说明 |
|------|------|
| `/注册 <名字>` | 注册狗牌 |
| `/改名 <新名字>` | 修改名字 |
| `/生日 yyyy.mm.dd` | 设置生日 |
| `/装备` | 列出当前装备 |
| `/装备 <装备名>` | 佩戴装备 |
| `/取下 <装备名>` | 取下装备 |
| `/装备补时 <装备名> <时长>` | 回溯装备已有时长 |
| `/送 @对方 <装备名>` | 赠送装备 |
| `/同意` | 接受赠送 |
| `/拒绝` | 拒绝赠送 |
| `/信息` | 查看个人信息卡片 |
| `/看 @群友` | 查看群友装备 |
| `/狗牌帮助` | 显示操作指南 |

## 配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `show_register_tips` | bool | true | 未注册时是否提示 |
| `birthday_reminder_enabled` | bool | true | 生日当天显示祝福 |
| `custom_name_prefix` | string | "" | 名字统一前缀 |
| `max_name_length` | int | 20 | 名字最大长度 |
| `max_equipment` | int | 0 | 同时佩戴上限（0=不限） |
| `max_equip_name_length` | int | 6 | 装备名字最大长度 |
