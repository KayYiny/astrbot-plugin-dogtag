# AstrBot Dog Tag 插件 🐶

注册管理狗牌身份，管理名字、生日和装备。

## 功能

- **身份注册** — 创建你的狗牌
- **信息管理** — 管理名字、生日、年龄
- **装备系统** — 佩戴/取下/回溯时长，自动计时
- **历史纪录** — 每种装备的最长佩戴时间
- **装备赠送** — 群友间互相赠送装备

## 安装

将本仓库所有文件放入 AstrBot 插件目录（`addons/plugins/`），重启 AstrBot。

## 命令

| 命令 | 说明 |
|------|------|
| `/注册 <名字>` | 注册狗牌 |
| `/改名 <新名字>` | 修改名字 |
| `/我的诞生日 yyyy.mm.dd` | 设置生日 |
| `/装备` | 列出当前装备 |
| `/装备 <装备名>` | 佩戴装备 |
| `/取下装备 <装备名>` | 取下装备 |
| `/装备补时 <装备名> <时长>` | 回溯装备已有时长 |
| `/送 @对方 <装备名>` | 赠送装备 |
| `/同意` | 接受赠送 |
| `/拒绝` | 拒绝赠送 |
| `/我的信息` | 查看个人信息卡片 |
| `/帮助` | 显示操作指南 |
